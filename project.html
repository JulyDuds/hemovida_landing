Sobre o Desenvolvimento

Uma solução completa para gestão de doações de sangue

O projeto HemoVida foi desenvolvido como parte do CET Tecnologias e Programação de Sistemas Informáticos, com foco especial na UFCD 5085 Criação de estrutura de base de dados em SQL.

Nosso objetivo foi criar uma plataforma robusta que integrasse todas as etapas do processo de doação de sangue, desde o cadastro de doadores até a distribuição para hospitais.

Estrutura do Banco de Dados

O coração do sistema HemoVida

SQL como Tecnologia Central

O banco de dados foi projetado seguindo as melhores práticas de normalização para garantir:

  • Integridade dos dados - Chaves primárias e estrangeiras bem definidas
  • Desempenho - Índices estratégicos para consultas frequentes
  • Segurança - Controle de acesso em diferentes níveis
  • Escalabilidade - Estrutura preparada para crescimento
